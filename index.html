DataSmith2 1.0 – clean up and export text

First stable version of DataSmith2. Load files, apply basic filters and export your cleaned text to TXT, CSV or JSON.

⬇ Download DataSmith2 1.0 ⭐ View source
Version 1.0 is free to use. A more advanced 2.0 and Pro edition are planned as separate products.

Features in 1.0

Everything you need to start working with text files.

  • Select single file or folder
  • Drag & Drop support
  • Basic filters (trim, remove empty lines, etc.)
  • Original / transformed preview
  • Export to TXT, CSV, JSON

Limitations

DataSmith2 1.0 is intentionally simple and focused.

  • No PDF / PNG / JPG export yet
  • No advanced presets or profiles
  • No multimedia (audio / video) features
  • UI is functional, not yet fully modernised
