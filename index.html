What is DataSmith2?

DataSmith2 is a lightweight desktop tool for transforming and cleaning file names in bulk. It provides instant preview, simple filters, and export options — all in a clean, fast UI. Built with Avalonia for Windows, Linux, and macOS.

Features

  • Remove numbers, lowercase, uppercase, spaces, or symbols
  • Add a custom prefix (invalid characters are automatically removed)
  • Drag & drop files or load entire folders
  • Live preview of original vs transformed names
  • Export to TXT, CSV, or JSON
  • Cross‑platform — works everywhere

Roadmap

Planned for version 2.0:

  • Modernized UI
  • Export to PNG, JPG, PDF
  • Persistent settings
  • Additional filters and automation tools

Build from Source

Requirements:

  • .NET 8 SDK
  • Avalonia 11.x

Build:

dotnet build

Run:

dotnet run
