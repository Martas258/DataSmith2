What is DataSmith2?

DataSmith2 is a lightweight desktop tool for transforming and cleaning file names in bulk. It gives you instant preview, simple filters, and export options — without scripting or complex regex.

Built with Avalonia and .NET, designed for Windows, Linux, and macOS.

Main features

  • Remove numbers, lowercase, uppercase, spaces, or symbols
  • Add a custom prefix (invalid characters are automatically removed)
  • Drag & drop files or load entire folders
  • Live preview of original vs transformed names
  • Export to TXT, CSV, or JSON
  • Cross‑platform desktop app

Roadmap

Planned for version 2.0:

  • Modernized UI
  • Export to PNG, JPG, PDF
  • Persistent settings
  • Additional filters and automation tools

Build from source

Requirements:

  • .NET 8 SDK
  • Avalonia 11.x

Build:

dotnet build

Run:

dotnet run

See the GitHub repository for detailed instructions and release notes.
